Quản lý đỉm

Ḥ th́ng chỉ quản lý lưu trữ đỉm ḱt quả học ṭp sinh vin cúi m̃i học kỳ và ḱt quả tích lũy của m̃i sinh vin. Cng tác quản lý quá trình của hoạt đ̣ng học ṭp như kỉm tra thi, thi lại, khng thục phạm vi ḥ th́ng này. Ḥ th́ng cho phép chuyn vin đào tạo thực hịn các chức năng import dữ lịu đỉm và in ḱt quả học ṭp theo các bỉu m̃u qui định.

1        Quản lý đỉm học kỳ

Chuyn vin đào tạo có th̉ quản lý đỉm trung bình học kỳ từng mn học của từng sinh vin thng qua Chức năng sinh vin/ ĐTB Học kỳ đ̉ chỉnh sửa hoặc tạo mới từng dòng dữ lịu

Màn hình quản lý đỉm trung bình học kỳ

 

Màn hình tạo mới đỉm trung bình học kỳ cho sinh vin

Thng thường chức năng này chỉ thực hịn khi có hịu chỉnh dữ lịu từ phản h̀i sai sót của sinh vin.

2        Quản lý file dữ lịu đỉm học kỳ

Chuyn vin đào tạo import dữ lịu đỉm học kỳ từ chức năng Dữ lịu Import/File dữ lịu đỉm sinh vin

Chức năng này quản lý các file dữ lịu đỉm học kỳ đã upload ln ḥ th́ng.

 

Màn hình quản lý file dữ lịu đỉm học kỳ

 

Sau khi upload file ln ḥ th́ng, chọn file này và thực hịn chức năng Import dữ lịu đỉm học kỳ
